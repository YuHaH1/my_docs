# 单元测试

## Jest单测

[文档地址](https://jestjs.io/docs/)

安装

~~~shell
pnpm add --save-dev jest
# 如果是ts
pnpm add --save-dev @babel/preset-typescript pnpm add --save-dev ts-jest
~~~

初始化

~~~shell
npx ts-jest config:init
~~~
